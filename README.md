# EpuckProject
The purpose of this project is to develop an e-puck robot capable of autonomously navigating an 8×8 maze environment. The robot starts near the red wall, which represents the starting point, and must successfully reach the green wall, which represents the goal.

To navigate through the maze, the robot uses the right-hand rule (right wall-following algorithm), allowing it to follow the wall on its right side while avoiding obstacles and making navigation decisions. The project also focuses on improving movement smoothness, navigation accuracy, and the overall stability and efficiency of the robot’s behaviour within the maze environment.
